# Proactive Recall

Research project: **proactive memory**—surfacing the right intention at the right moment (time, place, device, digital context), not only on demand.

## Scope (working)

- Trigger families: scheduled time, calendar-adjacent windows, location/venue, wearable and phone surfaces, digital workflow context (apps, sites).
- Memory layer: how personal notes (“jots”) are represented, ranked, and explained when a nudge fires.
- Trust and UX: cooldowns, snooze, transparency (“why now?”), avoiding notification fatigue.

## Related

Sibling exploration in the **Jot** codebase (`../jot`): local-first capture, workflow watcher, timed reminders, and LLM-assisted organization on macOS.
